# References — DreamWeave

Dossier de référence pour la spec de découpage et de style narratif.
Claude lit ces fichiers directement pour construire les specs.

## Structure

```
lightnovels/   → chapitres en texte brut (.txt ou .md)
webtoons/      → chapitres en PDF (impression depuis navigateur ou conversion CBZ→PDF)
spec/          → specs générées par Claude à partir des références
```

## Format recommandé

### Light Novels
- `.txt` ou `.md`
- 1 fichier = 1 chapitre
- Nommage : `[titre]-ch[num].txt` (ex: `solo-leveling-ch01.txt`)
- Source : copier-coller depuis Kindle / web / epub

### Webtoons
- `.pdf`
- 1 fichier = 1 chapitre (10-30 panels)
- Nommage : `[titre]-ch[num].pdf`
- Méthode recommandée :
  - Navigateur → Ctrl+P → "Enregistrer en PDF" (sur webtoon.com, tapas.io)
  - Ou : CBZ → PDF via Calibre (gratuit)

## Idéal : même arc dans les deux formats
Avoir le chapitre 1 d'une œuvre en LN + sa version webtoon
pour voir exactement comment le texte a été découpé en cases.

## Décisions prises
- `bg-popover` (fond solide) obligatoire sur tout DialogContent contenant des images — `glass` (backdrop-filter) crée un layer GPU composite qui dégrade le rendu des images enfants
- Pattern `onNodeUpdated` : callback passé du LoreGraphView au LoreNodeSheet pour mise à jour immédiate de rfNodes sans attendre le cycle React Query. À appliquer systématiquement pour toute mutation critique visible dans le graph
- Type picker sur le badge image (bas-gauche) plutôt que dans le header — cohérent avec le design two-column, moins encombrant
- `backdrop-blur-sm` interdit sur les nœuds React Flow — crée N layers GPU qui dégradent le rendu du canvas entier

## Ce qui a été livré (session 2 — polish)

### FloatingEdge — couleur dorée
- **Bug racine** : CSS `.react-flow__edge-path { stroke: #b1b1b7 }` (React Flow) écrasait l'attribut SVG `stroke="#F59E0B"` (priorité basse)
- **Fix** : passage à `style={{ stroke: "#F59E0B" }}` (inline = priorité maximale)
- **Glow** : suppression injection DOM SVG (fragile) → CSS `drop-shadow` sur `<g>` (fiable)
- Au repos : `drop-shadow(0 0 2px rgba(245,158,11,0.6))` — fil discret, pas néon
- Au hover : double drop-shadow intensifié

### Prévention connexions dupliquées
- `isValidConnection` bloque toute tentative de créer A→B si B→A ou A→B existe déjà

### Zone invisible anti-chevauchement
- `MIN_NODE_GAP = 60` autour de chaque carte
- Drag stop : snap-back si dépôt en chevauchement (via `dragStartPosRef`)

### Prévisualisation connexion (GoldenConnectionLine)
- `ConnectionLineComponentProps` importé
- Recalcul du centre du nœud depuis la position du handle bottom (`nodeCenterY = fromY - NODE_CARD_H/2`)
- Direction optimale calculée via dx/dy (identique FloatingEdge) → sortie par le bon côté
- Fil pointillé doré avec CSS drop-shadow

### Réorganisation intelligente (runAutoLayout)
- **Ordre** : Événement → Lieux → Personnages → Objets (haut → bas)
- **Espacements** : ROW_Y 0/300/600/900px, H_SPACING 210px
- **Tri barycentriques** : ordonne les nœuds par position moyenne de leurs voisins → minimise les croisements
- **Ressort horizontal** (SPRING=0.28, 120 iter) : aligne horizontalement les nœuds connectés
- **Post-processing 5 passes** : détecte et écarte les nœuds intermédiaires qui bloquent les connexions cross-row (fils qui sautent ≥ 2 rangs)

## Phase 1 — Sécurité (CRITIQUE) ✅ COMPLÉTÉE

### 1.1 Corriger la vérification JWT dans l'Edge Function ✅
- **Fichier** : `supabase/functions/generate-asset-image/index.ts`
- **Problème** : `getSubFromJwt()` décodait le JWT avec `atob()` sans vérifier la signature.
- **Résolution** :
  - [x] Remplacement par `verifyUserFromToken()` qui appelle `supabase.auth.getUser(token)` via l'endpoint `/auth/v1/user`
  - [x] Retour erreur 401 si le token est invalide ou expiré
  - [x] Suppression de la fonction `getSubFromJwt()` manuelle

### 1.2 Rate limiting & quotas de génération ✅
- **Fichiers** : Migration SQL + Edge Function + hooks frontend
- **Résolution** :
  - [x] Migration SQL : `supabase/migrations/20260213220000_add_usage_table.sql` (table `usage`)
  - [x] Migration SQL : `supabase/migrations/20260213230000_add_plan_to_profiles.sql` (champ `plan` sur profiles)
  - [x] Edge Function : comptage usage mensuel avant chaque génération
  - [x] Rejet HTTP 429 si quota dépassé (20/mois Free, 300/mois Pro)
  - [x] Enregistrement dans `usage` après chaque génération réussie
  - [x] Frontend : `useUserPlan()` hook avec compteur temps réel
  - [x] Dashboard : barre de progression usage + badge tier

### 1.3 Restreindre les CORS ✅
- **Fichier** : `supabase/functions/generate-asset-image/index.ts`
- **Résolution** :
  - [x] CORS dynamique via variable d'environnement `ALLOWED_ORIGIN`
  - [x] Fallback `"*"` si non défini (dev), configurable en production
  - [ ] **ACTION REQUISE** : Définir `ALLOWED_ORIGIN` dans les secrets Supabase en production

### 1.4 Ajouter des confirmations de suppression ✅
- **Fichiers** : `src/components/project/AssetLibrary.tsx`, `src/pages/Projects.tsx`
- **Résolution** :
  - [x] `AlertDialog` ajouté avant suppression d'assets et projets
  - [x] Messages clairs avec nom de l'élément et avertissement irréversibilité

### 1.5 Nettoyer le Storage lors de la suppression d'assets ✅
- **Fichier** : `src/services/storage.ts`, `src/services/assets.ts`
- **Résolution** :
  - [x] `deleteAssetImages()` récupère toutes les URLs et supprime via `supabase.storage.remove()`

### 1.6 Activer le safety checker FAL.ai ✅
